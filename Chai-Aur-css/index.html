Chai CSS to Inline Style

Apply classes to see inline styles!

Applied Inline Styles:

Features & Syntax

Colors & Backgrounds

Text Color: chaiT-{color}-{shade} (e.g., chaiT-blue-500) or chaiT-white

Background: chaiBg-{color}-{shade} (e.g., chaiBg-red-400) or chaiBg-black

Spacing (Padding & Margin)

Padding: Standard [prefix]-{px} or Custom [prefix]-{val}-{unit}
Prefixes: chaiP, chaiPx, chaiPy, chaiPt, chaiPb, chaiPl, chaiPr. (e.g., chaiP-2-rem)

Margin: Standard [prefix]-{px} or Custom [prefix]-{val}-{unit}
Prefixes: chaiM, chaiMx, chaiMy, chaiMt, chaiMb, chaiMl, chaiMr. (e.g., chaiMx-auto)

Typography

Font Size: chaiFs-{px} (e.g., chaiFs-24)

Font Weight: chaiFw-{weight} (e.g., chaiFw-bold, chaiFw-600)

Text Align: chaiTa-{align} (e.g., chaiTa-center, chaiTa-right)

Flexbox

Display Flex: chaiFlex

Direction: chaiFlex-direction-{row|col}

Justify: chaiFlex-justify-{start|center|end|between|around|evenly}

Align: chaiFlex-align-{start|center|end|stretch}

Gap: chaiFlex-gap-{1..12}

Layout & Size

Dimensions: chaiW-{val}, chaiH-{val}, chaiW-{val}-{unit} (e.g., chaiW-100-%, chaiH-50-vh)

Display: chaiD-{block|hidden|inline|inline-block}

Borders & Effects

Border: chaiB-{width}, chaiB-{color}-{width} (e.g., chaiB-2, chaiB-black-2)

Border Radius: chaiBr-{px} (e.g., chaiBr-10)

Shadow: chaiS-{s|m|l|xl|inner} (e.g., chaiS-xl)

Cursor: chaiC-{pointer|...} (e.g., chaiC-pointer)
